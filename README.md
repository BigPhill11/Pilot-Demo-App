# Pilot-Demo-App

A financial literacy web application built with React, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies

- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/)

## Editing the Code

**Local IDE** — Clone the repo, make changes, and push.

**GitHub** — Navigate to any file, click the pencil icon, edit, and commit.

**GitHub Codespaces** — Open the repo, click Code → Codespaces → New codespace.

## Deployment

This project can be deployed to any static hosting platform (Vercel, Netlify, etc.) by running:

```sh
npm run build
```

The output will be in the `dist/` folder.
